# PC Setup Bootstrap

这是一个用于新电脑初始化的交互式脚本项目。

## 一行命令执行

将下面命令里的 `<owner>` 和 `<repo>` 替换为你的 GitHub 用户名和仓库名：

```bash
curl -fsSL https://raw.githubusercontent.com/<owner>/<repo>/main/bootstrap.sh | bash
```

## 项目地址

- 仓库：`https://github.com/<owner>/<repo>`
- Pages：`https://<owner>.github.io/<repo>/`
